# 确定性评分

| 维度 | 分值 |
|---|---:|
| 大周期趋势 | 0–20 |
| 支撑阻力强度 | 0–25 |
| K 线确认 | 0–15 |
| 成交量确认 | 0–15 |
| 技术指标确认 | 0–15 |
| 盈亏比与上方空间 | 0–10 |

80–100 为 `strong_buy_candidate`，65–79 为 `buy_candidate`，50–64 为 `watch`，0–49 为 `no_trade`。硬性阻断条件可把候选降级为 `watch` 或 `no_trade`，但不修改明细以美化结果。新闻情绪当前为 `not_available`，不加减分。

区域触碰按冷却间隔去重；连续相邻 K 线只算一个触碰事件。有效反弹加分，有效跌破/突破扣分。多周期合并取同类行为的最大值而不是求和，避免同一次价格行为被重复累计。
